# bigfish


## [VimFx][5]的轻量级版本

- 安装

    先装[GreaseMonkey][1]，再点击：[vimfx-lite.user.js][6]

- 快捷键列表（不区分大小写）

    |按键|功能|
    |:---:|:---:|
    |`J`|向下滚动|
    |`K`|向上滚动|
    |`H`|向左滚动|
    |`L`|向右滚动|
    |`U`|向上翻页|
    |`M`|向下翻页|
    |`Y`|滚动到顶端|
    |`N`|滚动到底端|
    |`X`|关闭标签页|
    |`I`|进入屏蔽模式，忽略除`Esc`外所有快捷键|
    |`Esc`|解除屏蔽模式，或使输入框失去焦点|
